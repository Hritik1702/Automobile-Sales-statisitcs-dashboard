# Automobile Sales Analytics Dashboard

An interactive data analytics dashboard built using **Python, Dash, Plotly, and Pandas** to analyze automobile sales trends across years, recession periods, vehicle types, and economic indicators.

This project also includes **industry-style evaluation metrics** for demand analysis and decision support.

---

## Project Motivation

Automobile companies need to understand:
- Sales trends over time
- Impact of recession and unemployment
- Effectiveness of advertising spend
- Demand stability for inventory planning

This dashboard provides **visual analytics and statistical insights** to support these decisions.

---

## Features

- Yearly automobile sales analysis  
- Recession-period focused insights  
- Vehicle-type-wise sales and advertising analysis  
- Interactive dashboard (dropdown-based)
- Evaluation metrics:
  - Average Sales
  - Standard Deviation
  - Coefficient of Variation (CV)
  - Year-over-Year (YoY) Growth
  - Advertising Efficiency

---

## Tech Stack

- **Language:** Python  
- **Libraries:** Pandas, Plotly, Dash  
- **Styling:** Tailwind CSS  
- **Dataset:** IBM Skills Network Automobile Sales Dataset
